### 6000.2.11f1 Release Notes

#### Improvements

- URP: Added logging for SDR rendering configuration when 10-Bit HDR Output is active. This helps you understand what to expect and any automatic adjustments URP makes. Refer to \[Use Standard Dynamic Range \(SDR\) Rendering with HDR Output enabled\]\(https://docs.unity3d.com/6000.3/Documentation/Manual/urp/post-processing/hdr-output.html\) for more details.
    (UUM-115727)



#### Changes

- URP: Modified Universal Render Pipeline HDR Output behavior to maintain 16-bit HDR Output when switching to Render Pipeline Assets that don't support HDR Rendering.
    (UUM-115727)



#### Fixes

- 2D: Clean up and fix mesh and sprite normals for shadergraph.
