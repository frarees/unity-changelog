### 6000.3.11f1 Release Notes

#### Features

- License: Kerberos proxy authentication support.

- macOS: Added support for `CAMetalDisplayLink` on the Mac Player.<br>
    It is disabled by default and it can be enabled in the Player Settings by toggling "Use MetalDisplayLink \(Mac Player only\)". In alternative, you can also use the environment variable `UNITY_USE_METAL_DISPLAY_LINK=1`.<br>
    Using `CAMetalDisplayLink` decreases stuttering and improves frame pacing. It also makes `Time.deltaTime` more stable.
